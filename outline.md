# Copper & Grain Craft Distillery - Website Structure

## File Organization

### Core Files
- `index.html` - Main landing page with hero section and brand story
- `products.html` - Interactive product catalog with tasting notes
- `about.html` - Distillery story, process, and team information  
- `contact.html` - Contact form, location, and tour booking

### Resources Directory
- `resources/hero-distillery.png` - Main hero image of copper still
- `resources/products-collection.png` - Product showcase image
- `resources/logo.png` - Distillery logo
- `resources/aging-warehouse.png` - Warehouse background image

### JavaScript
- `main.js` - All interactive functionality and animations

## Page Content Structure

### Index.html - Landing Page
1. **Navigation Bar** - Logo, main menu, age verification
2. **Hero Section** - Large distillery image with typewriter tagline
3. **Brand Story** - Heritage and craftsmanship narrative
4. **Product Discovery Quiz** - Interactive component for recommendations
5. **Featured Products Carousel** - Premium spirits showcase
6. **Awards & Recognition** - Industry accolades
7. **Footer** - Contact info and social links

### Products.html - Product Catalog
1. **Navigation Bar** - Consistent across all pages
2. **Product Grid** - Interactive cards with hover effects
3. **Filter System** - By spirit type, flavor profile, price
4. **Tasting Notes Generator** - Interactive flavor profiling tool
5. **Virtual Distillery Tour** - Clickable hotspots with process info
6. **Purchase Information** - Where to buy, distribution

### About.html - Distillery Story
1. **Navigation Bar** 
2. **Heritage Timeline** - Founding story and milestones
3. **Production Process** - Step-by-step distilling journey
4. **Meet the Team** - Master distiller and key personnel
5. **Sustainability Practices** - Environmental commitment
6. **Facility Tour** - Virtual walkthrough of distillery

### Contact.html - Contact & Tours
1. **Navigation Bar**
2. **Contact Form** - Inquiry and tour booking
3. **Location Map** - Interactive distillery location
4. **Tour Information** - Schedule and booking details
5. **Private Events** - Corporate and special occasion hosting
6. **Wholesale Inquiries** - Distribution and partnership info

## Interactive Components Implementation

### Age Verification Gateway
- Modal overlay on first visit
- Smooth fade-in animation
- "Enter Website" button with copper glow effect

### Product Discovery Quiz
- 4-5 questions about taste preferences
- Smooth transitions between questions
- Results show personalized recommendations
- Progress indicator with amber liquid fill animation

### Virtual Distillery Tour
- Interactive map with clickable areas
- Hotspots for stills, barrels, bottling line
- Hover effects reveal equipment details
- Information panels with production process

### Tasting Notes Generator
- Dropdown menus for flavor descriptors
- Visual flavor wheel with clickable segments
- Generates shareable tasting cards
- Save multiple tasting sessions

## Visual Effects & Animations

### Scroll Animations
- Elements fade in as user scrolls
- Staggered reveals for product cards
- Parallax effects on hero sections

### Hover Effects
- Copper glow on buttons and links
- Image zoom with overlay information
- 3D tilt effects on product cards

### Background Effects
- Subtle particle system suggesting distillery atmosphere
- Warm lighting effects throughout
- Consistent copper and amber color accents
